# aquarium-temperature-monitoring-system

##題目發想緣起123
看到家裡的魚缸開關燈、餵食、控溫....等等 好麻煩，乾脆把這些功能都把它整合起來，做成一套系統。
在這套系統裡可以遠端(如:網頁、手機app)監控(ph值、溫度、鹽度、水位...等等)
也可以控制溫度的高低、餵食的時間、開關燈的時間、CO2打氣量的時間
